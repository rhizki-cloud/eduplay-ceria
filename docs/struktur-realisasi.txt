EduPlay Ceria - Struktur Realisasi Setelah Redesign

1. Tampilan publik responsif
- Desktop: sidebar, topbar, konten maksimal 1280 px, dashboard perkembangan.
- Tablet: grid adaptif dua kolom.
- Mobile: navigasi bawah lima menu dan tombol ramah sentuhan.

2. Modul aktif
- Beranda interaktif: misi harian, poin, level, streak, skor terbaik, lanjut belajar, aktivitas terbaru.
- Ruang Belajar: Numerasi kelas 4-6, Pola Hidup Sehat, dan Karakter.
- Zona Bermain: Aritmatika, Makanan Sehat, Cerita Karakter, Memory, Mencocokkan, Tebak Gambar, Urutan Angka, dan Kuis Pengetahuan.
- Prestasi: badge, piala, peringkat pribadi, riwayat, level, serta sertifikat cetak.
- Tentang aplikasi dan panduan pendamping.

3. Panel pengelola
- Login dan dashboard admin telah diperbarui agar responsif.
- CRUD materi dasar tetap tersedia.
- Folder modul admin lain dipertahankan sebagai fondasi pengembangan database berikutnya.

4. Penyimpanan
- Riwayat aktivitas disimpan di localStorage.
- Nilai dikirim ke MySQL melalui proses/prestasi/simpan.php saat database tersedia.
- PWA menggunakan service worker cache versi eduplay-ceria-v3.
